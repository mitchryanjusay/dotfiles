Setup guide

Check if +python is present in :ver if it's not present then:
Windows:
1. If you don't have Chocolatey, install Chocolatey first by executing the command below in an administrator level command prompt
 
@powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"

2. If you do have Chocolatey, execute the command below from an administrator level windows powershell

choco install editorconfig.core 

Update g:EditorConfig_exec_path to the executable path

if +python is present then comment out the following:
g:EditorConfig_exec_path
g:EditorConfig_core_mode
